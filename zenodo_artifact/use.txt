Intended Use and Limitations of the Artifact
================================================

Intended Use
------------
This artifact is designed to support research on Windows string comparison behaviors and Unicode security implications. Specifically:

1. The UMC tool demonstrates the conversion of Unicode modifier letters to base Latin characters, as described in Section 7.3 of the paper. 

2. The vulnerability analysis suite allows researchers to:
   - Explore the parameter space of the CompareStringW API (Section 4.2)
   - Identify characters that CompareStringW treats as equivalent (Section 4.2)
   - Discover programs potentially affected by inconsistent string comparison (Section 4.2)

3. The evaluation cases (Section 6) provide concrete examples for testing and validation.

Limitations
-----------
1. System Requirements: The tools are built for Windows systems and require specific API availability. They may not function correctly on non-Windows systems or older Windows versions.

2. Scope of Analysis: The program scanner is designed to analyze Windows binaries. Results may vary across different system configurations and software versions.

3. Performance: Full system scans (e.g., of System32 directory) are computationally intensive and may take considerable time to complete.

4. Version Specificity: The tools were developed and tested on Windows 10 with Visual Studio 2013. Behavior may differ on other configurations.

Ethical Use
-----------
This artifact is provided for academic research purposes only. Users should:
- Obtain proper authorization before scanning systems they don't own
- Respect software licenses and terms of use
- Handle any discovered vulnerabilities responsibly through appropriate disclosure channels

The provided evaluation cases are sufficient for reproducing the paper's results without scanning live systems.
